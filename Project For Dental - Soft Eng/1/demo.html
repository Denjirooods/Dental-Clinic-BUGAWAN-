The Macvil Dental Clinic Inventory Management System is a comprehensive web-based solution designed specifically for dental clinics. It features a modern, professional interface inspired by Google Material Design principles.

Installation Guide Full Documentation

System Requirements

Minimum Requirements

  • OS: Windows 7+, macOS 10.12+, Ubuntu 16.04+
  • Python: 3.7 or higher
  • RAM: 2GB available
  • Storage: 100MB free space
  • Browser: Chrome 60+, Firefox 55+, Edge 79+

Recommended

  • OS: Windows 10/11, macOS 11+, Ubuntu 20.04+
  • Python: 3.9 or higher
  • RAM: 4GB or more
  • Storage: 500MB free space
  • Browser: Latest Chrome, Firefox, or Edge

Quick Start

Windows Users

  1. Install Python from python.org
  2. Double-click run_system.bat
  3. Open browser to http://localhost:5000
  4. Login with admin/admin123

macOS/Linux Users

  1. Install Python 3
  2. Run ./run_system.sh
  3. Open browser to http://localhost:5000
  4. Login with admin/admin123
